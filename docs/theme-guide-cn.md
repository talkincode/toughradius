# ToughRADIUS 主题使用指南

## 快速开始

ToughRADIUS v9 提供了精心设计的浅色和深色两种主题模式，让您在不同的使用场景下都能获得最佳的视觉体验。

### 切换主题

1. 登录系统后，在顶部导航栏右侧找到主题切换按钮（太阳/月亮图标）
2. 点击按钮即可在浅色和深色主题之间切换
3. 系统会自动记住您的选择，下次访问时自动应用

## 自定义主题（开发者）

### 修改主题颜色

编辑 `/web/src/theme.ts` 文件：

```typescript
// 修改浅色主题的主色调
const lightPalette = {
  primary: {
    main: "#your-color-here",
    // ...
  },
};
```

### 添加自定义组件样式

```typescript
export const createAppTheme = (mode: PaletteMode) => {
  return createTheme({
    // ... 其他配置
    components: {
      MuiYourComponent: {
        styleOverrides: {
          root: {
            // 自定义样式
          },
        },
      },
    },
  });
};
```

### 在组件中使用主题

```tsx
import { useTheme } from "@mui/material";

const MyComponent = () => {
  const theme = useTheme();
  const isDark = theme.palette.mode === "dark";

  return (
    <Box
      sx={{
        // 使用主题变量
        backgroundColor: theme.palette.background.paper,
        color: theme.palette.text.primary,
        // 根据主题模式条件渲染
        border: isDark ? "1px solid #444" : "1px solid #ddd",
      }}
    >
      内容
    </Box>
  );
};
```

## 常见问题

### Q: 主题切换后页面没有变化？

A: 刷新浏览器页面，主题设置会自动保存并加载。

### Q: 如何重置为默认主题？

A: 清除浏览器的 localStorage 或在开发者工具中删除 `RaStore.preferences.theme` 键。

### Q: 能否设置系统跟随操作系统主题？

A: 当前版本需要手动切换，未来版本会支持自动跟随系统主题设置。

### Q: 某些第三方组件没有适配主题？

A: 可以在 `theme.ts` 中为该组件添加自定义样式覆盖。

## 技术细节

### 主题持久化

主题选择存储在浏览器的 localStorage 中：

- 键：`RaStore.preferences.theme`
- 值：`'light'` 或 `'dark'`

### 过渡动画

所有主题相关的颜色变化都使用 CSS 过渡：

```css
transition: background-color 0.3s ease, color 0.3s ease;
```

### 响应式设计

主题在所有设备尺寸上都能正常工作：

- 移动端（xs）
- 平板（sm, md）
- 桌面（lg, xl）
