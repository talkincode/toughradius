## 命名规则说明

文件命名格式：`rfcXXXX-主题描述.txt`

- `rfcXXXX` - RFC 编号
- `-` - 分隔符
- `主题描述` - 简短的英文主题描述（使用连字符分隔单词）
- `.txt` - 文件扩展名

### 主题描述缩写说明

- `vsa` - Vendor-Specific Attributes (厂商特定属性)
- `eap` - Extensible Authentication Protocol (可扩展认证协议)
- `nas` - Network Access Server (网络接入服务器)
- `ipv6` - Internet Protocol version 6
- `coa` - Change of Authorization (授权变更)
- `6rd` - IPv6 Rapid Deployment (IPv6 快速部署)
- `teap` - Tunnel Extensible Authentication Protocol (隧道可扩展认证协议)
- `ieee802` - IEEE 802 标准（包括 Wi-Fi）
