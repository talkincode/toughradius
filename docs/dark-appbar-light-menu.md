# 深色导航栏 + 浅色菜单图标配置

## 设计说明

为了提供更好的视觉对比和用户体验，ToughRADIUS v9 采用了以下设计策略：

### 🎨 固定配色方案

1. **顶部导航栏 (AppBar)**

   - **始终使用深色背景** (#1e293b - 深灰蓝色)
   - **浅色文字和图标** (#f1f5f9 - 浅灰色)
   - 无论浅色/深色主题，导航栏保持一致的深色风格

2. **侧边栏菜单 (Sidebar)**
   - **始终使用深蓝色背景** (#1e40af - 深蓝色)
   - **浅色菜单项和图标** (rgba(255, 255, 255, 0.85))
   - 提供良好的视觉层次和对比度

## 配色优势

### ✨ 视觉一致性

- 顶部导航栏和侧边栏在任何主题下都保持一致的外观
- 用户不会因为切换主题而感到迷失

### 👁️ 高对比度

- 深色背景 + 浅色图标/文字 = 优秀的可读性
- 符合无障碍设计标准 (WCAG)

### 🎯 专业外观

- 深色导航区域传达专业、稳重的感觉
- 适合企业级应用的视觉定位

## 技术实现

### 主题配置 (theme.ts)

```typescript
MuiAppBar: {
  styleOverrides: {
    root: {
      // AppBar 始终使用深色背景
      backgroundColor: '#1e293b',
      color: '#f1f5f9',
      borderBottom: '1px solid rgba(148, 163, 184, 0.2)',
    },
  },
},

MuiDrawer: {
  styleOverrides: {
    paper: {
      // 侧边栏始终使用深蓝色背景
      backgroundColor: '#1e40af',
    },
  },
},

RaMenuItemLink: {
  styleOverrides: {
    icon: {
      // 菜单图标始终使用浅色
      color: 'rgba(255, 255, 255, 0.85)',
    },
  },
},
```

### CustomAppBar 组件

```tsx
<AppBar
  sx={{
    backgroundColor: "#1e293b",
    color: "#f1f5f9",
  }}
>
  {/* 图标和按钮也使用浅色 */}
</AppBar>
```

### CustomMenu 组件

```tsx
<Box
  sx={{
    backgroundColor: "#1e40af",
    color: "#ffffff",
  }}
>
  {/* 菜单项自动继承浅色样式 */}
</Box>
```

- ✅ 侧边栏：深蓝 (#1e40af)
- ✅ 主内容区：浅色背景 (#f8fafc)

### 深色主题

## 颜色规范

| 元素       | 背景色                 | 文字/图标色            | 用途                         |
| ---------- | ---------------------- | ---------------------- | ---------------------------- |
| 顶部导航栏 | #1e293b                | #f1f5f9                | 应用标题、设置按钮、主题切换 |
| 侧边栏     | #1e40af                | rgba(255,255,255,0.9)  | 导航菜单                     |
| 菜单图标   | -                      | rgba(255,255,255,0.85) | 菜单项图标                   |
| 激活菜单   | rgba(59,130,246,0.4)   | #ffffff                | 当前选中的菜单项             |
| 悬停菜单   | rgba(255,255,255,0.12) | #ffffff                | 鼠标悬停的菜单项             |

## 交互效果

### 菜单项状态

1. **默认状态**

   - 背景：透明
   - 文字：rgba(255, 255, 255, 0.9)
   - 图标：rgba(255, 255, 255, 0.85)

2. **悬停状态**

   - 背景：rgba(255, 255, 255, 0.12)
   - 文字：#ffffff
   - 图标：#ffffff + 放大 5%

3. **激活状态**
   - 背景：rgba(59, 130, 246, 0.4)
   - 文字：#ffffff (加粗)
   - 图标：#ffffff

### 导航栏按钮

- 主题切换按钮：悬停时旋转 180°
- 设置按钮：悬停时放大 5%
- 统一使用浅色图标 (#f1f5f9)

## 设计理念

这种固定配色方案的设计理念是：

1. **减少视觉噪音**：导航区域保持一致，让用户专注于内容区
2. **品牌识别**：深蓝色侧边栏成为应用的视觉标识
3. **主题独立**：导航栏不随主题变化，提供稳定的导航体验
4. **对比分明**：深色导航 + 动态内容区，层次清晰
